Proyectos

Durante más de un año, trabajé en Toledo desarrollando páginas en Blazor y programas en Windows Form en .NET Framework 4.7 y .NET Core.

Además, participé en un proyecto clave donde desarrollamos un backend para una API en .NET Core con MySQL como base de datos, para un negocio de venta de chatarra.

En este proyecto, manejamos remitos, facturación, cuentas corrientes, roles y tokens de validación de usuario, siguiendo principios SOLID y una arquitectura de software sofisticada.

Invito a explorar más detalles en mi GitHub desde el pie de página de mi página web.
